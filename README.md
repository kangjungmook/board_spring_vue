## 프로젝트의 목적
 Spring Boot 3와 Vue.js를 익히기 위해 게시판을 만들었습니다

## 기술
* Java: JDK 21 <br>
* Spring Boot: 3.0<br>
* Frontend: Vue.js<br>
* IDE: IntelliJ IDEA, Visual Studio<br>
* Database: MySQL

## 기능
* 회원가입
* 로그인(JWT 토큰 방식)
* 게시물 수정
* 게시물 조회
* 게시물 작성
* 게시물 삭제
* URL 패턴 제한
